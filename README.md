# mvn-repo

Add repository in pom.xml
______
```
<repositories>
  <repository>
    <id>mvn-repo-mvn-repo</id>
    <url>https://raw.github.com/TennisBoyyy/mvn-repo/mvn-repo/</url>
    <snapshots>
      <enabled>true</enabled>
      <updatePolicy>always</updatePolicy      
    </snapshots>
  </repository>
</repositories>
```
